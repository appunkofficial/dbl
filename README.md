# Sistem Data 
Sistem manajemen data warga digital untuk kelurahan dengan fitur penyimpanan lokal dan ekspor data.

## Fitur Utama

✅ **Input Data Warga**  
- Formulir dengan validasi lengkap
- Input data pribadi + dokumen pendukung
- Preview dokumen sebelum upload

📁 **Manajemen Data**  
- Penyimpanan lokal di browser (localStorage)
- Pencarian data cepat
- Sorting data berdasarkan kolom
- Hapus data dengan konfirmasi

📄 **Ekspor Data**  
- Export ke format PDF
- Generate laporan terstruktur
- Header dan footer profesional

🔍 **Fitur Tambahan**  
- Pencarian real-time
- Preview data lengkap
- Tampilan responsif
- Animasi modern

## Teknologi

- HTML5
- CSS3 (dengan Bootstrap 5)
- JavaScript ES6
- LocalStorage
- jsPDF (untuk export PDF)
